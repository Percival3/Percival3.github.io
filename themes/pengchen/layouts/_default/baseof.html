#### 第三步：复刻侧边栏 (sidebar.html) 把你的 React `Sidebar` 代码翻译成 Hugo 模板。注意把 `{SITE_DATA.xxx}` 变成 `{{ .Site.Params.xxx }}`。 http://googleusercontent.com/immersive_entry_chip/0 #### 第四步：定义内容页面模板 (single.html 和 list.html) React 里的 `ContentArea` 逻辑现在变成了 Hugo 的标准页面渲染。 **1. 列表页模板 (用于 Blog/Posts 列表)** `themes/pengchen/layouts/_default/list.html` http://googleusercontent.com/immersive_entry_chip/1 **2. 单篇文章模板 (用于显示文章详情)** `themes/pengchen/layouts/_default/single.html` http://googleusercontent.com/immersive_entry_chip/2 #### 第五步：更新你的配置文件 (hugo.toml) 最后，我们需要把你的 `SITE_DATA` 数据填入 `hugo.toml`，并配置菜单。 http://googleusercontent.com/immersive_entry_chip/3 ### 接下来你需要做什么？ 1. **创建对应的内容文件**： * 创建 `content/about.md` (复制你 React 代码里 `SITE_DATA.about` 的内容进去)。 * 创建 `content/social.md`。 * 在 `content/posts/` 下创建你的文章。 2. **运行**： * 在终端输入 `hugo server`。 * 打开 `localhost:1313`。 **区别解释：** 原来的 React 网站是点击按钮 -> JS 切换内容 (不刷新)。 现在的 Hugo 网站是点击按钮 -> 浏览器跳转到新页面 (刷新，但在本地极快)。 外观上，它们将**完全一模一样**，但现在的版本更适合写博客、管理大量文章，并且对 SEO 极其友好。
