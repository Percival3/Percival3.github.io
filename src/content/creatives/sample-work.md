## 概述

这是一个示例创意作品条目，用于展示作品展示页面的布局效果。

## 技术细节

- 使用 Canvas 2D API 绘制粒子系统
- Perlin Noise 生成平滑的流场向量
- 支持亮色/暗色主题自动切换粒子配色
- 鼠标悬停产生引力/斥力交互效果

## 灵感来源

灵感源自自然界中的流体运动与数学中的向量场可视化。
